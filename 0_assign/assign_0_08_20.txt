Question 4:
    a] -> Identify the longest open reading frame (ORF) in the following DNA sequence (note: start codon ATG, stop codons TAG, TGA, TAA) ATGATATGATGTCACAGGATGTAGATGCTCCGACATAAGCTTA
    b] -> Pick one position in the ORF found above and indicate which type of mutation (insertion, deletion, or mutation) would be called a "frame-shift mutation". Briefly explain why

Answer:
    a]: 'ATGATGTCACAGGATGTAGATGCTCCGACATAA'
    b]:
        - Insertion mutation in between indices (6, 17) in above ORF would create a frame-shift mutation causing the seq to be shorter than the original one.
        - For example, 'ATG ATG GTC <C>AC AGG ATG TAG', it was cut short because 'C' was inserted and as a result, TAG codon was read.
